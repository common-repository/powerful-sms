Plugin para enviar notificação por SMS após fazer pedidos usando WooCommerce e outras integrações do Woocommerce

== Description ==

Este plugin não tem nenhuma relação com o provedor de serviço de SMS ou qualquer outro plugin que se integra.

Com este plugin você pode enviar mensagens automáticas por SMS integrando com diversos plugins Wordpress com o Woocomerce

- Reconquiste clientes enviando um cupom por sms automáticamente quando o pedido for cancelado;
- Crie mensagens personlizadas com mais de 30 variáveis disponíveis;
- Envie o código de rastreio automaticamente com o link para acompanhar;


Serviço de SMS disponível:
* Disparo PRO

Integrações disponíveis:
* Woocommerce
* Powerful Auto Chat

== Installation ==

1. Faça upload da pasta `powerful-sms-wp` para `/wp-content/plugins/` diretorio
2. Ative o plug-in através do menu "Plugins" no WordPress.
3. Compre uma chave e insira no local indicado. 
4. Ative os gatilhos que desejar e personalize o teamplate de cada um.
5. Faça um teste criando um pedido .enjoy
